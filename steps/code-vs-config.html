Code vs Config

Code: Use common code patterns to configure plugins.

Config: Use common "flags" to configure plugins.

It's about you and your work; Style + task.

Config: when done properly is easier to work with. Documentation is the key to success here, otherwise, it's no different than code. Code: Requires understanding of the language syntax and it's idosyncracies but is more often the more powerful. Difficult for beginners.
